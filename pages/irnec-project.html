IRNEC Project

First Clinical Deployment: Real-World Impact

Back to Documentation

Collaborators

Context

Necrotizing enterocolitis (NEC) is a gastrointestinal emergency in premature newborns. Registry data is scarce due to fragmented documentation in unstructured neonatal records.

Objective

Building the first automated NEC registry pipeline. We extract structured clinical variables from records to advance research into risk factors, treatment, and prevention.

Methodology

Co-development with neonatologists using The Architect, The Extractor, and the DLP. Bound to the Zurich region for strict compliance (nFADP/GDPR).

Project Snapshot

Disease Necrotizing Enterocolitis
Population Premature and neonates in critical care
Data Source Unstructured neonatal records
Languages German · French · English
Variables 850 total (159 treated)
Tools The Architect · Extractor · DLP
Output Validated research-ready dataset
Status Active — pilot in progress
